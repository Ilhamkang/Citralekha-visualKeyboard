# Citralekha-visualKeyboard


This keyboard provides special Latin characters to be used in the transliteration and transcription system of Indonesian manuscript and inscriptions. The code is based on [Ganesh Kavar's On Screen Keyboard Python Project](https://github.com/ganeshkavhar/On-Screen-Keyboard-Python-Project).
